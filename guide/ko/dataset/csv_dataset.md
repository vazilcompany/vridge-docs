데이터 추가
------


'파일 선택' 란을 클릭하거나, 파일을 드래그하여 업로드 할 수 있습니다. 

'데이터베이스 선택' 란을 클릭하여, 외부 데이터베이스에 있는 데이터를 가져 올 수 있습니다.<br> 
현재 연결 가능한 DB 목록
- MariaDB
- MongoDB
